# Topwar Updates

This is where I post diffs whenever Top War pushes a config or locale update.

Each file covers what changed between two versions — new rows added to tables, removed entries, modified values, new locale strings, etc. When a config and locale update drop at the same time I combine them into one file.

Files are named by date (e.g. `April 3 2026.txt`).

---

## What's in each file

**Config diff** — compares two versions of the game's internal config. Shows:
- New/removed tables
- Tables with row count changes
- New IDs added to key tables (heroes, skins, items, buffs, etc.)
- Full row data for anything new

**Locale diff** — compares two versions of the English locale file. Shows:
- New keys and their values
- Removed keys
- Any keys that had their text changed

---

## Version format

- Config versions look like `8366`, `8458`, etc.
- Locale versions look like `3348`, `3358`, etc.

These are the version numbers the game uses internally — not the app store version.
